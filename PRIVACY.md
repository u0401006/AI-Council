# 隱私權政策 Privacy Policy

**最後更新：2024 年 12 月**

AI Council（以下稱「本擴充功能」）尊重並保護使用者隱私。本政策說明本擴充功能如何收集、使用及保護您的資料。

---

## 資料收集

### 我們收集的資料

本擴充功能僅收集以下資料，且全部由使用者主動觸發：

| 資料類型 | 說明 | 儲存位置 |
|---------|------|---------|
| API 金鑰 | 使用者輸入的 OpenRouter 及 Brave Search API 金鑰 | 本地端 `chrome.storage.sync` |
| 使用者設定 | 模型選擇、提示詞設定、介面偏好 | 本地端 `chrome.storage.sync` |
| Context 項目 | 使用者透過右鍵選單加入的選取文字或頁面內容 | 本地端 `chrome.storage.local` |
| 對話紀錄 | 使用者的查詢與 AI 回應歷史 | 本地端 `chrome.storage.local` |

### 我們不收集的資料

- 個人識別資訊（姓名、電子郵件、地址等）
- 瀏覽記錄或網頁造訪紀錄
- 使用者活動（點擊、滑鼠移動、按鍵記錄）
- 位置資訊
- 財務或付款資訊
- 健康資訊

---

## 資料使用

收集的資料僅用於以下目的：

1. **API 金鑰**：向 OpenRouter 及 Brave Search 發送 API 請求
2. **使用者設定**：記住您的偏好設定，跨裝置同步
3. **Context 項目**：作為 AI 查詢的附加背景資料
4. **對話紀錄**：讓您回顧過往查詢

---

## 第三方服務

本擴充功能使用以下第三方 API：

### OpenRouter (openrouter.ai)
- 用途：發送 LLM 查詢請求
- 傳送資料：使用者輸入的問題、附加的 Context 內容
- 隱私權政策：https://openrouter.ai/privacy

### Brave Search (api.search.brave.com)
- 用途：網頁搜尋功能（選用）
- 傳送資料：搜尋關鍵字
- 隱私權政策：https://brave.com/privacy/browser/

**重要提示**：當您使用本擴充功能發送查詢時，您的問題及 Context 內容會傳送至上述第三方服務。請勿在查詢中包含敏感個人資訊。

---

## 資料儲存與安全

- 所有資料僅儲存於您的本地裝置及 Chrome 同步儲存空間
- 本擴充功能不設有任何自有伺服器
- 本擴充功能不收集、儲存或傳送資料至開發者
- API 金鑰以 Chrome 內建的儲存機制保存，不會明文外洩

---

## 資料刪除

您可隨時刪除本擴充功能儲存的所有資料：

1. **刪除單一項目**：在側邊面板中刪除個別 Context 項目或對話紀錄
2. **清除所有資料**：移除擴充功能即可刪除所有本地儲存資料
3. **重設設定**：在選項頁面重設為預設值

---

## 權限說明

本擴充功能請求以下權限：

| 權限 | 用途 |
|------|------|
| `storage` | 儲存 API 金鑰、設定及對話紀錄 |
| `sidePanel` | 顯示主要操作介面 |
| `activeTab` | 取得當前分頁資訊以標記 Context 來源 |
| `scripting` | 擷取使用者選取的文字或頁面內容 |
| `contextMenus` | 建立右鍵選單功能 |
| 網站存取 | 向 OpenRouter 及 Brave Search API 發送請求 |

所有權限僅在使用者主動觸發功能時使用。

---

## 兒童隱私

本擴充功能不針對 13 歲以下兒童，亦不故意收集兒童資料。
