核心功能

結合多個 AI 模型的智慧，獲得更全面、可靠的答案

多模型並行查詢

同時向 GPT、Claude、Gemini、Grok 等模型發送查詢，即時串流顯示各模型回應。

匿名互評機制

各模型匿名評估其他模型的回答，產出客觀排名與評語，自動識別高品質回答。

Agent Planner

智能任務規劃系統，根據問題類型自動選擇最佳處理策略與工具組合。

Skills 專業模組

9 種專業技能模組：研究者、教育者、事實查核、創意寫作、技術問答等。

Web Search 整合

Brave Search API 整合，支援即時網路搜尋與 AI 延伸搜尋，獲取最新資訊。

Vision & Image

支援圖片輸入分析（Vision Mode）與 Gemini 圖片生成，視覺化創作無限可能。

快速開始

三步驟完成設定

1

安裝擴充功能

  1. Clone 或下載 GitHub repo
  2. 開啟 Chrome，前往 chrome://extensions/
  3. 開啟右上角「開發人員模式」
  4. 點擊「載入未封裝項目」
  5. 選擇 repo 資料夾
2

設定 API Keys

點擊擴充功能圖示 → 右鍵 → 選項

OpenRouter API Key 必要 取得 →
Brave Search API Key 可選 取得 →
3

開始使用

  • 選擇參與 Council 的模型（至少 2 個）
  • 選擇主席模型
  • 點擊擴充功能圖示開啟 Side Panel
  • 輸入問題，享受多 AI 智囊團服務
